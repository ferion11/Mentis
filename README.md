# Mentis
Linux subliminal messages App

* Fork from original and old Brazilian site:
https://web.archive.org/web/20071209131053/http://www.danilocesar.com/blog/projetos/mentis/

* Mentis read a file named ".mentis" in your home and display it on the screen. Like:
* ~/.mentis
<pre>
I love my neighbor
I love my enemies
I trust in God
</pre>
